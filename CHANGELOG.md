# Changelog

## v0.3.0 (2026-04-09)

### New Features
- **破解豆瓣反爬机制** — 自动求解 SHA-512 工作量证明挑战，电影评分/类型/导演/演员/海报均可正常获取
- **电影详情弹窗** — 点击推荐卡片弹出详情窗，展示完整信息（类型、导演、演员、可观看平台、豆瓣链接）
- **推荐卡片增强** — 卡片上直接显示导演信息

### Performance
- **并发抓取优化** — 从 3 并发 × 6 批次提升至 5 并发 × 15 批次，详情获取速度提升约 5 倍
- **按需平台搜索** — 平台可用性搜索从同步流程移至"为我推荐"按钮，同步过程不再卡住

---

## v0.2.0 (2026-04-09)

### Breaking Changes
- **移除 TMDB 依赖** — 不再需要任何外部 API Key，零配置即可使用

### New Features
- 电影元数据直接从豆瓣详情页获取（类型/导演/演员/评分/海报）
- 流媒体可用性通过各平台搜索页查询
- 爱奇艺改为海外版 iQIYI (iq.com)

### Bug Fixes
- 修复进度条百分比溢出问题（限制在 99% 以内）
- 修复系统代理导致 httpx 连接超时（`trust_env=False`）

---

## v0.1.0 (2026-04-09)

### Initial Release
- 豆瓣数据同步 — 自动爬取用户「看过」和「想看」列表
- 用户偏好分析 — 统计偏好类型、导演、演员
- 跨平台推荐 — 支持腾讯视频、爱奇艺、优酷、芒果TV、Netflix、Disney+、Max
- 智能排序 — 豆瓣评分 (50%) + 偏好匹配度 (50%)
- 暗色主题界面 — Vue 3 + TailwindCSS 单页应用
- 一键启动 — `start.bat` 双击即用
